# 介绍 
这是一个为了巩固小程序的练习项目，仿音乐小程序，采用了网易云音乐 NodeJS 版 API，使用微信小程序原生框架mina开发，完成了播放列表热门歌单排行榜MV等功能，其余功能尚在完善。
